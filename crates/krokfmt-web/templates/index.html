Installation

# Install via cargo cargo install krokfmt # Or download pre-built binaries from GitHub releases

Usage

# Format files in place krokfmt src/ # Check if files are formatted (CI mode) krokfmt --check src/ # Print formatted output without modifying files krokfmt --stdout file.ts
